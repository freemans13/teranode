Request blocks

query Parameters
offset
integer (PaginationOffset) >= 0
Default: 0

The number of records to skip before starting to collect the result set. With a limit of 50, an offset of 50 implies viewing the 2nd logical page of data.

limit
integer (PaginationLimit)
Default: 100

The maximum number of records to return. The value -1 represents no limit and signifies all records.
